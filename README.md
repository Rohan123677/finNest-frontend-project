# Project Title: 
FinNest - Premium Financial Investment Dashboard

# Description : 
This is a responsive FinNest Website frontend developed using only fundamental web technologies: HTML, CSS, and JavaScript. No external frameworks or libraries have been used. The project is designed to demonstrate core frontend development skills by creating a modern, user-friendly financial interface.

# Purpose : 
The main objective of this project is to strengthen frontend development skills by building a complete user interface without using any frontend frameworks such as Bootstrap, React, Angular, or Vue.js.

# How To Run: 

1. Clone the repository:
   bash
   git clone https://github.com/your-username/your-repository-name.git
   

2. Navigate to the project folder:
   bash
   cd your-repository-name
   

3. Open the project:
   - Open the project folder in Visual Studio Code or any code editor.

4. Run the project:
   - Open the index.html file directly in your web browser.
   - OR
   - Use the Live Server extension in Visual Studio Code:
     - Install the Live Server extension.
     - Right-click on index.html.
     - Select Open with Live Server.

5. The website will open in your default browser and you can explore the frontend pages.

# Requirements :

- A modern web browser (Google Chrome, Microsoft Edge, Mozilla Firefox, etc.)
- Visual Studio Code (recommended)
- Live Server extension (recommended for development)
